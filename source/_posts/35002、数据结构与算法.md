#### JS 数组的实现

```js
js数组优化：类似C语言的方式定义数组

C语言数组特点：
1、数组中必须存放相同的数据类型
2、申请数组才内存空间时，必须指定数组的大小

数组耗性能的操作：
1.数组的扩容：本来16个的放不下了，会创建一个扩大1倍（32）新数组
2.数组前面中间插入（删除）元素、性能也是非常低的，底层要一个个移动元素（元素位移）

但平时我们只是查数据：用索引（这个速度是很快的）
用大O表示就是：O(常数)
```

#### 算法性能越来越低

```js
2n²+2n + 8 ---> 2n²+2n+1 ---> 2n² ---> n² === O(n²)

O(1) // 常数
O(log(n)) // 对数
O(n); // 线性
O(nlog(n)) // 线性和对数乘积
O(n²) // 平方
O(2 n) // 指数
```

#### 数组与栈

```js
数组是一种线性结构，可以在任意位置插入和删除数据

栈和队列是比较常见的受限的线性结构

栈：后进先出的一种数据结构

面试题：
1、按654321进栈、哪些是不合理的出栈顺序
思路：一个元素想出来，那么它顶上必须没有元素

2、栈：解决括号匹配面试题：()()()())()())(((()
思路：遇到左括号进栈，遇到右括号，把栈顶元素取出来
```
